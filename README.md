# OBS Updater

This repository is a standalone version of the OBS Studio Updater, refactored to use MSBuild and vcpkg for dependency management instead of CMake.

ToDo:
- curl and HTTP/2 support

## Building

1. Follow the instructions here to set up vcpkg: https://learn.microsoft.com/en-us/vcpkg/get_started/get-started-msbuild?pivots=shell-powershell#1---set-up-vcpkg
2. Build solution in Visual Studio
